# Grand-Luxe-Hotel-system
A simple C++ console-based hotel management system with room selection and food ordering.

Overview:

This is a simple console-based Hotel Management System written in C++. The program allows users to select a room type and order meals (breakfast, lunch, or dinner) from different menus.

It is designed to demonstrate the use of nested switch statements, user input handling, and basic program flow control.

 Features:
 Room selection system:
1)Standard Room
2)Deluxe Room
3)Suite Room

Food ordering system:

1)Breakfast
2)Lunch
3)Dinner

Multiple menu options for each meal
Interactive user prompts
Instant order confirmation messages

 Technologies Used:
      C++
      
Standard Library (<iostream>)

 How It Works:
 Step 1: Select Room Type

User chooses a room:

1 → Standard Room ($120/night)
2 → Deluxe Room ($220/night)
3 → Suite Room ($450/night)

 Step 2: Select Meal Type

User selects:

1 → Breakfast
2 → Lunch
3 → Dinner

 Step 3: Choose Menu Item

Each room type has its own menu options.

Example:
Standard Breakfast:
1)Tea & Toast
2)Omelette
3)Pancakes

Deluxe Lunch:
1)Grilled Chicken
2)Qorma
3)White Karahi

Suite Dinner:
1)Italian
2)Soup
3)Chinese

 Step 4: Order Confirmation

The system displays a message like:

you have ordered pizza

 Concepts Used:
1)Nested switch statements
2)Conditional logic
3)User input/output (cin, cout)
4)Program flow control

 Limitations:
1) No billing system (total cost not calculated)
2) No input validation (invalid inputs may cause issues)
3) No loop (program runs only once)
4) Hardcoded menu and prices
5) No real hotel database
